3. Mathematical Framework & Core DSP Logic
The audio stream processing loops must execute downstream from the parametric EQ stages, but upstream from the Clipper and Limiter stages, using this strict sequence:
Module 1: De-Esser (Frequency-Selective Bandpass Sidechain)
Route a duplicate copy of the stereo input buffers into a sidechain Linkwitz-Riley 2nd-order bandpass filter centered at deesserFreq to isolate the sibilance envelope.
Rectify the sidechain signal to decibels: \(X_{\text{side\_dB}} = 20 \cdot \log_{10}(\max(\vert{}s_{\text{left}}\vert{}, \vert{}s_{\text{right}}\vert{}) + 10^{-5})\).
Compute sibilance gain reduction: If \(X_{\text{side\_dB}} > \text{deesserThresholdDB}\), then \(\Delta G_{\text{dB}} = \text{deesserThresholdDB} - X_{\text{side\_dB}}\), else \(\Delta G_{\text{dB}} = 0\).
Smooth \(\Delta G_{\text{dB}}\) using a fast attack (\(1.0 \text{ ms}\)) and release (\(50.0 \text{ ms}\)) coefficient. Convert the result to a linear multiplier \(G_{\text{linear}} = 10^{G_{\text{smoothed\_dB}} / 20}\).
Apply \(G_{\text{linear}}\) directly to the main audio channels: \(x[n] = x[n] \cdot G_{\text{linear}}\).
Module 2: Multiband Compressor (Linkwitz-Riley 4th-Order Splitter)
Split the incoming audio into three independent streams (Low, Mid, High) using cascaded Linkwitz-Riley 4th-Order (24 dB/oct) Low-Pass and High-Pass filters built from the mbCrossLowMid and mbCrossMidHigh frequencies.
For each split band stream, compute independent peak tracking envelopes:
\(X_{\text{band\_dB}}=20\cdot \log _{10}(\max (|\text{band}_{\text{left}}|,|\text{band}_{\text{right}}|)+10^{-5})\)
Apply static compression mappings if a band exceeds its specific threshold:
\(\text{If\ }X_{\text{band\_dB}}>\text{Threshold}_{\text{band\_dB}}:\Delta G_{\text{band\_dB}}=\left(\text{Threshold}_{\text{band\_dB}}+\frac{X_{\text{band\_dB}}-\text{Threshold}_{\text{band\_dB}}}{4.0}\right)-X_{\text{band\_dB}}\quad (\text{Fixed\ 4.0\ Ratio})\)
Smooth each band's gain reduction independently using fixed internal time constants optimized for those frequencies (Low: \(40\text{ms}\) attack / \(200\text{ms}\) release; Mid: \(20\text{ms}\) attack / \(100\text{ms}\) release; High: \(10\text{ms}\) attack / \(50\text{ms}\) release).
Convert to linear factors, multiply them against their respective isolated band audio channels, and sum the three updated streams back into a single stereo mix buffer before handing it to Module 3.
Module 3: Standard Compressor
Apply the primary wideband compression engine exactly as specified previously:
Track the unified rectified peak decibel level across channels: \(X_{\text{dB}} = 20 \cdot \log_{10}(\max(\vert{}x_{\text{left}}\vert{}, \vert{}x_{\text{right}}\vert{}) + 10^{-5})\).
Compute the attenuation delta: If \(X_{\text{dB}} > \text{compressorThresholdDB}\), then \(\Delta G_{\text{dB}} = \text{compressorThresholdDB} + \frac{X_{\text{dB}} - \text{compressorThresholdDB}}{\text{compressorRatio}} - X_{\text{dB}}\), else \(\Delta G_{\text{dB}} = 0\).
Smooth the gain envelope using calculated slider coefficients (\(\alpha _{\text{attack}}\), \(\alpha _{\text{release}}\)).
Convert back to a linear scale, factor in the compressorMakeupGainDB, and apply the multiplier to the stereo stream.
Module 4: Dynamic Range Expander (Downward Expander Mode)
Use the compressor's wideband decibel envelope (\(X_{\text{dB}}\)) as the sidechain input to determine expansion.
Compute downward expansion behavior when the signal drops below the threshold floor:
\(\text{If\ }X_{\text{dB}}<\text{expanderThresholdDB}:\Delta G_{\text{exp\_dB}}=(\text{expanderThresholdDB}-X_{\text{dB}})\cdot (1.0-\text{expanderRatio})\)
\(\text{Else}:\Delta G_{\text{exp\_dB}}=0\)
Clamp \(\Delta G_{\text{exp\_dB}}\) to prevent attenuation from dropping below your safety range floor: \(\Delta G_{\text{exp\_dB}} = \max(\Delta G_{\text{exp\_dB}}, \text{expanderRangeDB})\).
Smooth this expansion envelope using a quick attack parameter (\(5.0\text{ ms}\) to handle incoming signals quickly) and a smooth release parameter (\(200\text{ms}\)). Convert to a linear factor and multiply across both stereo channels.
