#### ✂️ Soft Clipper Algorithm
Let x be the driven input sample (x_in * Drive_linear), t be the linear threshold amplitude, and k be the Knee Smoothness value.
1. Calculate the boundary limits of the knee region:
 x_lower = t - (k / 2)
 x_upper = t + (k / 2)
2. Apply conditional processing sample-by-sample:
 - If |x| <= x_lower:
   f(x) = x
 - If |x| > x_upper:
   f(x) = sign(x) * t
 - If x_lower < |x| <= x_upper:
   f(x) = sign(x) * (x_lower + ((|x| - x_lower) - (|x| - x_lower)^2) / (2 * k))

#### 🛡️ Brickwall Limiter Algorithm
1. Peak Detection: Scan the look-ahead history buffer to find the absolute maximum peak sample value: p = max(|x_look_ahead|).
2. Target Gain Calculation: If p exceeds the linear Output Ceiling (c), target gain factor is: g_t = c / p. (If p does not exceed c, g_t = 1.0).
3. Envelope Smoothing: Compute the time constant coefficient alpha using the user's Release Time (tau in seconds) and the active system sample rate (currentSampleRate):
 alpha_release = exp(-1.0 / (tau * currentSampleRate))
4. In the processing loop, smooth the current gain coefficient (g_c) sample-by-sample:
 - If g_t < g_c: g_c = g_t (Instant Attack)
 - If g_t >= g_c: g_c = g_c * alpha_release + g_t * (1.0 - alpha_release)
5. Apply Gain: Multiply the delayed sample exiting the look-ahead buffer by the smoothed gain coefficient: y = x_delayed * g_c.

#### 📊 Gain Reduction Tracking
At the end of each buffer frame, compute the Gain Reduction decibel output and store it in a thread-safe variable accessible by the main thread:
Gain_Reduction_dB = 20 * log10(g_c)
